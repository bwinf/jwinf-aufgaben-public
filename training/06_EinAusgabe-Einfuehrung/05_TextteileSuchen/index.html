Textteile suchen

Schreibe ein Programm:

Das Programm soll eine Textzeile einlesen und das erste Auftreten von we suchen. Es soll die Position ausgeben.

Bitte schau dir vorab die Erläuterungen der Bausteine unter "weitere Hinweise" an.

Weitere Hinweise:

Mit dem Baustein wird das erste oder letzte Auftreten eines Suchbegriffs gesucht. Es wird die Position des Suchbegriffs zurück gegeben. Falls der Begriff nicht gefunden wurde, wird 0 zurück gegeben.

Über den kleinen Pfeil, können dir verschiedenen Optionen ausgewählt werden.

Das Programm soll eine Textzeile einlesen und das letzte Auftreten von we suchen. Es soll die Position ausgeben.

Das Programm soll die Textzeilen einlesen und abwechselnd die Position des ersten und letzten e's ausgeben.

Beachte: Dein Programm muss mit allen Testfällen zurechtkommen.
